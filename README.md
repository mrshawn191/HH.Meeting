## HH.Meeting

This is an education platform that connects users who wants to learn similar things together. 

## Dependencies used
- .NET Framework 4.5
- AspNet.WebApi
- AspNet.WebApi.Client
- AspNet.WebApi.Core
- EntityFramework
- SimpleInjector (DI)
- Serilog
- Azure WebJobs
- Azure ServiceBus
- NewtonsoftJson
- NUnit
- Moq
